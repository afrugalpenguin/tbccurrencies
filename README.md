# TBCCurrencies

A World of Warcraft TBC Anniversary Classic addon that adds a **Currency** tab to the Character frame, giving you a complete overview of all your TBC currencies in one place.

## Features

- Adds a 5th tab to the Character frame (alongside Character, Reputation, Skills, PvP)
- Displays all TBC currencies organized by section:
  - **Money** - Gold/Silver/Copper
  - **PvP** - Honor Points, Arena Points
  - **PvE** - Badge of Justice
  - **Battleground Marks** - WSG, AB, AV, Eye of the Storm
  - **World PvP** - Halaa tokens, Spirit Shards
  - **Faction Tokens** - Glowcap, Honor Hold/Thrallmar marks, Arcane Rune, Holy Dust, Sunmote
- Collapsible/expandable sections
- Zero-count currencies shown dimmed; non-zero shown in full color
- Faction-specific items filtered automatically (Alliance/Horde)
- Updates in real-time as currencies change

## Installation

Download from [CurseForge](https://www.curseforge.com/wow/addons/tbccurrencies) or extract into your `Interface/AddOns/` folder.

## Game Version

TBC Anniversary Classic (Interface 20505)
